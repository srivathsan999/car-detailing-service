# AutoShine - Car Detailing Service HTML Template

A premium, ThemeForest-ready HTML template for car detailing services with modern design, dark mode support, and comprehensive dashboard.

## Features

- ✅ **Fully Responsive** - Mobile-first design, works on all devices
- ✅ **Dark Mode** - Complete theme system with localStorage persistence
- ✅ **Modern UI** - Glassmorphism and Neomorphism effects
- ✅ **Chart.js Integration** - Interactive charts in dashboard
- ✅ **GSAP Animations** - Smooth, subtle animations
- ✅ **Complete Pages** - All frontend, auth, and dashboard pages included
- ✅ **ThemeForest Ready** - Professional code structure and presentation

## Tech Stack

- **HTML5** - Semantic markup
- **Tailwind CSS** - Utility-first CSS framework (via CDN)
- **Vanilla JavaScript** - No frameworks
- **GSAP** - Animation library
- **Chart.js** - Dashboard charts
- **localStorage** - Theme persistence

## Brand System

- **Accent**: #22D3EE

## Usage

1. Open `index.html` in a browser
2. Theme toggle is available on all pages
3. Dashboard is accessible without login (demo mode)
4. All links are functional
